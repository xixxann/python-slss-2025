## Loops

> Consider the code below.
> How many times does this iterate?

```python
for _ in range(5):
	# some code in here
```

This iterates 5 times.

> Consider this code, now.
> How many times do you think this iterates?

```python
for _ in range(5):
	for _ in range(5):
		# some code in here
```

This iterates 25 times.
